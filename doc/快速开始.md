# 布道师学习通 - 快速开始指南

## 环境要求

- JDK 8 或更高版本
- Maven 3.6 或更高版本
- MySQL 8.0 或更高版本
- Redis 6.0 或更高版本

## 第一步：初始化数据库

### 方式一：使用脚本（推荐）

```bash
# Windows
init-database.bat

# Linux/Mac
chmod +x init-database.sh
./init-database.sh
```

### 方式二：手动初始化

```bash
# 1. 创建数据库
mysql -u root -p
CREATE DATABASE budaos DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;
exit;

# 2. 导入数据
mysql -u root -p budaos < database/db.sql
```

## 第二步：配置应用

编辑配置文件：`budaos-application/src/main/resources/application.yml`

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/budaos?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
    username: root
    password: your_password

  redis:
    host: localhost
    port: 6379
    password: your_redis_password
```

## 第三步：编译并运行

### 方式一：使用启动脚本（推荐）

```bash
# Windows
start.bat

# Linux/Mac
chmod +x start.sh
./start.sh
```

### 方式二：手动编译运行

```bash
# 编译
mvn clean package -DskipTests

# 运行
java -jar budaos-application/target/budaos-monolith.jar

# 或者使用Maven插件
cd budaos-application
mvn spring-boot:run
```

## 第四步：访问系统

启动成功后，访问以下地址：

- **访问地址**: http://localhost:9080
- **后台管理**: http://localhost:9080/mgr
- **默认账号**: admin
- **默认密码**: admin123

## 常见问题

### 1. 编译失败

检查Maven配置和依赖下载情况：

```bash
mvn clean install -U
```

### 2. 数据库连接失败

确认MySQL服务已启动，并检查用户名密码配置。

### 3. Redis连接失败

确认Redis服务已启动，如果没有设置密码，将配置中的password留空。

### 4. 端口被占用

修改 `application.yml` 中的 `server.port` 为其他端口。

## 目录结构说明

```
budaos-monolith/
├── budaos-parent/          # 父POM（依赖管理）
├── budaos-common/          # 公共模块
│   ├── utils/              # 工具类
│   ├── common/             # 公共类
│   └── modules/            # 业务实体和接口
├── budaos-service/         # 业务服务层
│   └── modules/            # Service和Mapper实现
├── budaos-web/             # Web控制层
│   └── modules/            # Controller层
├── budaos-auth/            # 认证授权
├── budaos-application/     # 启动模块
└── database/               # 数据库脚本
```
